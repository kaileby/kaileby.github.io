线代

行列式

行列式的基本性质：

  • A 的转置的行列式等于 A 的行列式
  • 可将·某行/列的公因式提取出来
  • 行/列互换，行列式的值添负号
  • 行倍加，列倍加，行列式的值不变
  • 行列式两行成比例，行列式等于 0
  • 行列式可拆
    • 如何拆？

行列式计算 - 具体的

  • 行列式展开 (具体的)

  • 不能展开？

    • 根据行列式性质化简

      • 化为上三角、下三角

      • 化为拉普拉斯

        • 什么是拉普拉斯？

          公式

      • 范德蒙

      • 爪型

        • 化为上三角、或下三角
          • 具体做法：Ri=Ri - n*R1

      • 三对角线

        • 3,4阶 用行列式性质化简 成三角
        • n 阶 - 用 归纳法

行列式计算 - 抽象的

联系其它知识点，结合已知条件

  • 与矩阵的联系：

    • 秩 小于 未知数个数(列向量个数/列数) → 行列式等于0
    • 与可逆矩阵的联系
    • 与伴随矩阵的联系

  • 与向量的联系:

    • 向量与方程组是换两种研究角度，本质是一样的
    • 向量、方程组是分块矩阵的延伸

  • 与双特的联系

    • 行列式等于特征值的累乘 - （这是由行列式的展开推导出来的）
    • 找到其相似矩阵 （一般该相似矩阵为对角矩阵），跟找特征值是一个道理

行列式等于零的证明

联系其它知识点，结合已知条件

  • 与矩阵(方阵)的联系：
    • 秩 小于 未知数个数(列向量个数/列数) → 行列式等于0
    • 行列式对应的矩阵不可逆 → 行列式等于0
  • 与向量的联系:
    • 向量与方程组是换两种研究角度，本质是一样的
    • 向量、方程组是分块矩阵的延伸
  • 与方程组的联系：
    • Ax=0 有非零解 → 秩小于未知数个数 → 行列式等于0
  • 与双特的联系
    • 有行列式等于特征值的累乘 → 找出一个特征值为 0
    • 找到其相似矩阵 （一般该相似矩阵为对角矩阵），跟找特征值是一个道理

矩阵

矩阵运算的基本性质

  • 结合律：ABCD= A BC D
  • 大部分矩阵不满足交换律，除 A 与 A-1 ；A 与 A* 外

矩阵乘法

  • A 的 N次方

    思想：未知的东西，将其与已知的东西联系起来求

    • 将 A 拼成 α * β的形式，这样好算
    • 找A的相似矩阵，相似矩阵的 N 次方特别好求

求可逆矩阵

A*A-1 = E

  • 具体的矩阵直接初等行变换
  • 抽象的矩阵：定义法
  • 若是分块矩阵，求可逆，用公式
  • 联系伴随矩阵求可逆

求伴随矩阵

A * A*= |A| * E

  • 定义法
  • 联系可逆矩阵
关于伴随矩阵的秩的考察
  • R(A*)

初等矩阵的性质

  • 初等矩阵求可逆
    • 乘数 - 乘相反数
    • 换行 - 不变
    • 行倍某数加 - 行倍某数的倒数加

正交矩阵

A的转置 = A 的逆

  • 正交矩阵的性质的应用

    • 用在相似对角化求 Q矩阵中

      普通的矩阵求逆矩阵，通过行变换，或是已知条件给出的伴随矩阵求出，都挺麻烦的；

      而正交矩阵的逆矩阵特别好求；

      又 对于能相似对角化的矩阵来说，其 Q 的每个向量都是无关的；

      所以只需要把无关的矩阵化成正交矩阵 (正交化单位化即可)

  • 几何意义：向量垂直

秩

秩是连接 行列式 矩阵 向量 方程组 双特 的桥梁

向量 方程组 算是从矩阵分出去的，属于分块矩阵的一支

行列式 也是从方阵分出去的，也属于矩阵的一支

题目往往是用 秩 的思想得出结论去论证一些问题

  • 秩的性质

    • A 可逆， AB=C 或 BA=C

      有 R(B)=R(C)

      原理：

      • 可逆矩阵是由若干初等矩阵组成的
      • 对矩阵做若干次初等变换（相当于乘单位矩阵），秩不变

    • A * B = 0 ; A 为 m*n , B 为 n * s

      R（A）+ R(B) <= N

      原理：不知

      联系：

      • 齐次方程有解

    • AB=C ; A 为 m*n , B 为 n * s

      R(AB)=R(C)<= min( R(A) , R(B) )

      原理：不知

    • 记错了：R(A) + R(B) <= R(A+B)

      R(A+B) <= R(A) + R(B)

    • R(A | b) >= R(b)

      • 类似的 在行上增加也满足这个式子

    • R(AT A)=R(A)

      原理

      由 ATAx=0 与 Ax=0 有公共解证出；

      Ax=0 → ATA x=0 →R()

秩的联系

  • 秩与行列式的联系

    • 秩的定义本就是 N 阶子式的行列式不等于 0，自然具有联系

      • R(A)< N → N阶子式等于0

  • 秩与矩阵的联系

    秩与行列式有联系

    行列式与可逆有联系

    → 秩与 可逆矩阵有联系

    • R(A)< N → N阶子式等于0 → A 不可逆

  • 秩与向量、方程组的联系

    向量与方程组是密不可分的

    • 秩与相关、无关的联系

      • R(A) < N →其次方程有非 0 解 → A 的列向量相关
      • R(A) = N →其次方程只有 0 解 → A 的列向量无关

    • 秩与线性表出的联系

      • R(A|b) = R(A) = N → 非齐次方程有唯一解 → b 能被A的列向量线性表出→（A,b）线性相关→A无关→b能被 A唯一表出

      • R(A|b) = R(A) < N → 非齐次方程有无穷解 → b 能被A的列向量线性表出→（A,b）线性相关→A中存在一个极大无关组能表出b

向量

相关、无关的证明

  • 反证法

    反证法是所有证明题的通用法则

    • 假设 “不能” 为 “能”，推导出与已知条件的矛盾，即证明成功

  • 定义法

    • 乘 某物，某物取决于是否能乘出 0

  • 用秩的观点

    • 前面已经提到 秩与 相关无关的联系

  • 凑 AB=0

    还是用秩的观点

  • 与其次方程有无非 0解的联系

相关无关的性质考察

  • 低维无关 → 高维无关

    低维无关=R(A)=n,由 R(A)<=min(m,n)得知，n<m

    添加行。A变成 m+s * n的矩阵，s自然小于 m+s →R(A)=n,即高维无关

  • 低列相关 → 高列相关

    低列相关= R(A)<n,给A 添列，A变成了 m * n+s 的矩阵，R(A)自然小于n+s，所以高列相关

线性表出的证明

  • 定义法
  • 用秩的观点 非齐次方程有解

向量组表出

  • 向量组单向表出

    AB=C

    Ax=b1 Ax=b2 … Ax=bn 有解；即这 N 个方程有公共解

  • 向量组双向表出，即等价

    与矩阵等价的区别练习

    矩阵等价与向量组等价本无联系。

    • 向量组等价可以看成是：

      A 等价 B ,可以看成是 A 右乘一个 m*n 的矩阵X 得到了B，不要求 m=n

      若恰巧 m=n，且此时 X 还可逆，则就说矩阵A等价矩阵B，这样的情况，也符合向量组等价的定义

    • 矩阵等价

      是 A 左乘 或 右乘 一个可逆矩阵，得到 B,说 A B矩阵等价

方程组

讨论方程组有无解的问题

方程组有公共解

  • 得出两方程组的基解
  • 设存在 k1 … kn ，存在 m1 … mn，有 k1x1 + … knxn=m1y1+… mnyn
  • 解由基解组成的方程组，求出 k1… kn ,m1…mn

方程组有同解

特征值、特征向量

双特的计算 -具体

  • |x-A|=0，求出特征值 （考察了行列式）
  • （x-A）c=0，求特征向量 （考察了解方程）

求双特 - 抽象

  • 定义

  • 找相似矩阵

    由已知推导未知

  • 特征值公式

    思想：右已知推导未知

对于特殊的矩阵

  • 秩为1的矩阵
    • 一个特征值等于迹，其余特征值为 0

判断矩阵相似

  • 矩阵相似的必要条件
    • 行列式： 各自特征值的累积要相等
    • 矩阵：R(A)=R(B)
    • 双特：特征值相等
    • 迹相等
  • 当前 4 个都满足的话，就通过 相似的传递性来判断

求矩阵相似对角化

  • 可相似对角化的充要

    • 有 n 个无关的特征向量

  • 可相似对角化的充分

    • 有 n 个不等的特征值

      or

    • n 重跟有 n 个无关的特征向量

求使得矩阵相似对角化的矩阵 Q

最好求可逆矩阵的矩阵是 正交矩阵

正交矩阵的每个列向量都是无关的，但 无关不一定正交

无关矩阵 化为 正交矩阵 （即正交化 单位化）

二次型

二次型化为标准型

二次型对应的矩阵是个 实对称矩阵

标准型对应的矩阵是个 对角矩阵

  • 由 实对称矩阵必可 化为对角矩阵

→ 二次型 必可化为 标准型

所以 二次型化标准型就是，在求矩阵对角化

判断合同

  • 合同的充要

    • 两矩阵正、负惯性系数相等，合同
    • QT A Q=B,Q 可逆，合同

  • 合同、相似的区别

    相似比合同要求更严，相似不仅要求正负惯性指数相等，更要求特征值相等

正定矩阵的判定

  • 定义： xT A x >0
  • 正定矩阵必定对称
  • 正定矩阵的充要
    • N 阶顺序主子式都大于0
    • xT A x >0
  • 正定矩阵的充分
    • N 阶顺序主子式都大于0 → 行列式大于 0
    • aii 都大于0
    • 特征值都大于0 →正惯性指数等于n
