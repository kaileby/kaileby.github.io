hexo 主题配置

发表于 2019-06-28 更新于 2020-06-07 分类于 博客相关

搭建好 hexo 博客之后，界面是相当简陋的，接着就要自定义想要的样式，装修毛胚房。

配置文件 _config.yml

  • 博客站点下的_config.yml

    这个配置文件位于本人的 D:\Project\Blogs 目录下诸如，该配置文件可用于设置博客的总体的搭配，使用何种样式、部署到哪里、以及其它后续需要加入的自定义模块等，如下所示的几个配置选项：

    • 设置 RSS 订阅

      我们常常会有订阅别人文章的需求，有更新的时候希望能有提醒的功能，RSS就是这样一个订阅的方式

      tips： 还有第二步，打开 theme 下的_config.yml，找到 rss：，接着配置：rss: /atom.xml

  • 主题 Theme下的_config.yml

    这里以 next 主题为例，该配置文件位于本人的D:\Project\Blogs\themes\next 目录下，该配置文件决定了博客的大部分样式，以下给出几个非常关键的配置选项:

    • Schemes 设置主题的风格

    • sidebar 侧栏设置

      侧栏设置包括：侧栏位置、侧栏显示与否、文章间距、返回顶部按钮等等

    • Sidebar Avatar 博客头像设置

  • 设置 RSS

    我们常常会有订阅别人文章的需求，有更新的时候希望能有提醒的功能，RSS就是这样一个订阅的方式
